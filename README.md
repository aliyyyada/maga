# maga — тренажёр к экзаменам

Веб-тренажёр для подготовки к вступительным экзаменам в магистратуру. Открой
[`index.html`](index.html) в браузере — вопросы и правильные ответы уже
встроены в файл, сервер не нужен.

## Что внутри

- **index.html** — тренажёр: одиночный/множественный выбор, сопоставление,
  порядок; режимы "все вопросы", "только ошибки", случайная подборка;
  прогресс сохраняется в localStorage браузера.
- **pi.md** — банк вопросов по направлению «Программная инженерия»
  (09.04.04), в структурированном markdown (вопрос / варианты / правильный
  ответ), из которого сгенерированы данные для `index.html`.

Исходные PDF со статистикой ответов по остальным направлениям в репозиторий
не включены (большие файлы, хранятся только локально).

## Как добавить ещё один экзамен

1. Разбери вопросы из PDF в `.md`-файл по тому же формату, что и `pi.md`.
2. Попроси собрать из него данные и добавить как новый режим в `index.html`.
