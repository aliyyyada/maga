# maga — тренажёр к экзаменам

Веб-тренажёр для подготовки к вступительным экзаменам в магистратуру. Открой
[`index.html`](index.html) в браузере — вопросы и правильные ответы уже
встроены в файл, сервер не нужен.

## Что внутри

- **index.html** — тренажёр на 4 направления (446 вопросов): выбор
  направления, режимы "все вопросы" / "только ошибки" / случайная подборка;
  одиночный и множественный выбор, сопоставление, порядок; прогресс по
  каждому направлению сохраняется отдельно в localStorage браузера.
- **pi.md** — «Программная инженерия» (09.04.04), 153 вопроса.
- **прикладнаяИнформатика090403.md** — «Прикладная информатика» (09.04.03), 75 вопросов.
- **РазвитиеИТПрод090402.md** — «Развитие ИТ-продуктов» (09.04.02), 141 вопрос.
- **фиит020402.md** — «ФИИТ» (02.04.02), 77 вопросов.

Все `.md`-файлы — структурированный markdown (вопрос / варианты с `[x]` на
правильных / правильный ответ), из которого сгенерированы данные для
`index.html`. Исходные PDF со статистикой ответов в репозиторий не включены
(большие файлы, хранятся только локально).

## Как добавить ещё один банк вопросов

1. Разбери вопросы из PDF в `.md`-файл по тому же формату, что и уже
   существующие банки.
2. Попроси собрать из него данные и добавить как новое направление в
   `index.html`.
